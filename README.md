Overview

This project is a simple Python-based login system that restricts repeated incorrect password attempts. After a fixed number of failed login attempts, the system temporarily locks access before allowing the user to try again. It demonstrates basic authentication and security concepts.

Features
User login with username and password
Limits failed login attempts
Temporarily locks access after multiple failures
Simple command-line interface
Beginner-friendly Python project
Requirements
Python 3.x
How to Run
Clone this repository.
Open the project in Jupyter Notebook or any Python IDE.
Run the Python file.
Enter the username and password when prompted.
If the maximum number of failed attempts is reached, the system locks access temporarily.
Example
Enter username: admin
Enter password: 1111
Invalid credentials
Attempts left: 2

Enter username: admin
Enter password: 12345
Login successful
